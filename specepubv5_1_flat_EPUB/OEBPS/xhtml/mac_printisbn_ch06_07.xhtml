6.5. Anchor (Hyperlinks and Cross References)

Link titles should include meaningful information. E.g., “More about Jane Eyre” is useful to the user of assistive technology; “More” is not.

  • Link text clearly indicates destination, or meaningful alternative is provided in a title attribute
  • Avoid redundant links.
  • Do not rely on color as the only indicator of the existence of a link: also use underlining or bold (strong).
  • A secondary means of identifying linked text has been provided
  • All cross references (“Chapter 9”, “see Chapter 1”, “in Module 2”, etc) should be hyperlinked. Schematron will look for these cases with the following terms:
    • Chapter
    • Part
    • Module
    • Figure
    • Table
    • Equation number (eq-number)
    • Example
  • All links must be hyperlinked.
  • Backlinks require the role="doc-backlink". These should be added to hyperlinks in Glossaries, Indexes, back of book Extended Descriptions, and Answer sections which point back to the main content.
