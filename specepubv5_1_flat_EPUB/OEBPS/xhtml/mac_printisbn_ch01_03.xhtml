1.2. Outputs from the Vendor

  • Specifications and Standards:EPUB follows Macmillan Learning EPUB specifications and meets WCAG 2.0 AA Accessibility Compliant Reflowable EPUB 3.2 file standards. NOTE: Until EPUB version 3.2 is officially approved and accepted by EPUB Check, the OPF metadata should still declare 3.0 as the version, even though the file meets 3.2 standards.
  • Copyright page is updated to remove printing information from digital version:
    • Remove Printing and Binding line, identifying print book manufacturer(s).
    • Remove manufacturing line, e.g. "Manufactured in the United States of America" or "Printed in China"
    • Note: Do not remove print key, e.g.
      • 1 0 9 8 7 6
      • f e d c b a
    • If the print ISBN is present inside the CIP data, update the line to include the EPUB version ISBN, e.g.
      • Replace ISBN line with: ISBN 978-1-936221-55-4 (print) | ISBN 978-1-5660-508-5 (EPUB)
      • Remove any other print book ISBN(s) from the copyright page and replace with EPUB ISBN only, e.g.:
        • ISBN 978-1-319-05851-7 (Paperback)
        • ISBN 978-1-319-06829-5 (Loose-leaf Edition)
        • Remove both lines and replace with: ISBN 978-1-319-06827-1 (EPUB)
  • Semantics: The EPUB file is semantically structured, tagged, and organized according to all of the standards included in this specification. Furthermore, copyright 2019 titles going forward will have a validation via Schematron in order to verify that the file was correctly semantically structured.
  • Validation: The EPUB achieves “Passed” status within the current version Macmillan Learning EPUB Validator tool. See 7 Content QA for more details. Acceptable Validator warnings have been checked, confirmed OK, and communicated to Macmillan Learning, per Validator tool instruction document.
  • Accompanying cover JPEG: A separate JPEG of just the front cover should be created and it should be named using the EPUB3 ISBN-13 followed by “_FC“.
  • Delivery: Successful delivery of the outputs will be expected. See separate document “EPUB File Delivery Instructions for Vendor” for details.
  • Content: Unless otherwise instructed, the EPUB should reflect final pass content, which may include late-stage corrections made after printer proof review and after production of any sample chapter EPUB.
