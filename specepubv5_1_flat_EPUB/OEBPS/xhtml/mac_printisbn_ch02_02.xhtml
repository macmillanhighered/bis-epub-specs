For all titles created or remediated after the publication of this specification that also conform to Macmillan Schema, four additional meta tags should be included to facilitate Schematron validation.

  • meta property with name book_type and value of either part or chapter
  • meta property with name schema and value of the schema version (2, 3, 3.1, etc)
  • meta property with name process and value of either "production" or "final"
  • The schema on all XHTML pages must
    • Match each other
    • Match the OPF
    • Is the most recent schema, if it is a new title
  • meta property with the name profile and a non-empty value. When the file is a flat EPUB, the value must be “flat”. When the file is an enhanced EPUB, the value must be “enhanced”.

For ALL Macmillan titles, insert the following Conformance metadata into the EPUB3 file. 

        <meta property="dcterms:conformsTo"> EPUB Accessibility 1.1 - WCAG 2.2 Level AA </meta>
