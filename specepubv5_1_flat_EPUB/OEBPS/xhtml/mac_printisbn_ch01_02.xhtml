1.1. Inputs from Macmillan Learning

  • Print files for conversion:
    • Application files: Final InDesign Application Files, including all InDesign linked files (i.e., hi res images). Note: When the application files originate in LaTeX format, the EPUB conversion should be performed from the LaTeX files direct to EPUB. (i.e., No InDesign files are available.)
    • Print PDF files for reference
      • Optional: Special instructions/book treatments (Print PDF marked with additional EPUB conversion instructions)
    • Covers/Components: Front cover, inside front cover, inside back cover, back cover, and any inserts documents to be included within the EPUB.
    • Images with Credits in place and optimized for EPUB inclusion.
  • CSS: Copyright 2019 titles and beyond will be using Macmillan EPUB CSS with title specific color palette. No special design instructions should be considered.
  • Required Metadata:
    • Alt Text (Short and Long Description) Microsoft Excel Document
    • Book Metadata to be included in the EPUB: --title, subtitle, edition, creator (author), publisher (Macmillan Learning or BFW High School), subject (discipline), rights (copyright year), date (published date), identifier (EPUB ISBN), source (print book ISBN), and type = education (see <dc:type>education</dc:type> in 2.1. OPF Metadata Coding Sample below for placement)
    • Accessibility Metadata to be included in the EPUB: Refer to section 6.1 Discovery-Enabled EPUB Publications for the list of required accessibility metadata values.
    • Educational Metadata to be included in the EPUB:
