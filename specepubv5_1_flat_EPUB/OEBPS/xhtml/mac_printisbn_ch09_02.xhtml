9.1. Floats

The classes .left and .right may be applied to certain elements in order to float them to the left or right, respectively. (Please refer to the tagging guidelines for more on this).

  • Elements that appear before or after <ul> or <ol> elements should never have a left float applied to them. If a float is requested, the element must float to the right. Otherwise, the floated element will interfere with the rendering of the bullets or numbers in the list.
  • Elements containing MathML elements as children should never have a float applied to them.
  • Aside and numbered figures should never have a float applied to them.
