Dataset 

Students or instructors can click on a dataset and download one or many files specific to that dataset. 

Supported File types:

CSV (.csv): these open in a new tab within Safari on a Mac (press Command/Control + S to save it) 

Word

PDF

Excel (.xls, .xlsx)

JMP (.jmp)

Mac-Text (.txt)

Minitab (.mtw, .mtp)

PC-text (.txt)

R (.rda)

SPSS (.por)

TI-Calc (.8Xm, .8XI, .89m, .89I)

Datasets can be partially automated. Some hand work is still required but the initial load of conversion of images can be automated. 

If the user is offline, the widget will not work.
