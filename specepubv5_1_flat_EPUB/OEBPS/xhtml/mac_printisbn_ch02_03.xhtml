2.2. EPUB Profiles

An EPUB profile is an output typing referring to the content inside the EPUB package. There are currently four profiles, each defined in the Schema:

  • Flat: An EPUB package conforming to the EPUB3 Specification
  • Enhanced: An EPUB package containing enhancements created with Javascript and iframes
  • Read and Practice: An EPUB package containing enhancements created with Javascript and iframes, but with differing elements
  • Amazon: An EPUB package conforming to the EPUB3 Specification and to Amazon conversion guidelines
