6.2. Titles

Label epub:type ARIA role Description
Label label   The text label that precedes a ordinal in a component title (e.g., 'Chapter', 'Part', 'Figure', 'Table').
Ordinal ordinal   An ordinal specifier for a component in a sequence of components (e.g., '1', 'IV', 'B-1').
Subtitle subtitle doc-subtitle An explanatory or alternate title.
Title title   The primary name of a work, section or component.
