Flat File Download Link

A widget that allows flat files that are placed on our internal servers to be linked out from ePub pages. If the flat file is not called out in narrative, Media Production must be given explicit instructions on where to place the link to the file, and any wording that should accompany it. If the user is offline, the file will not download.
