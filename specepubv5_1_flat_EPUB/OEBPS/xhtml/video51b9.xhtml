Video and Multivideo

Video

Videos that have been uploaded to Brightcove can be placed in an ePub using this widget. If videos are not called out in narrative, Media Production must be given explicit instructions on where to place videos, and any wording that should accompany them.

If the user is offline, the videos will not play.

Multivideo

Multiple videos that have been uploaded to Brightcove can be placed in the same widget so users can watch groupings of videos. Videos are listed in a left-hand side bar and users can click to play each. These videos are accessed via an icon that brings up this version of the video player. If videos are not called out in narrative, Media Production must be given explicit instructions on where to place videos and any wording that should accompany them.

If the user is offline, the videos will not play.
